====================================
Compilation et Exécution du jeu
====================================

Librairies requises: SDL2

1) Aller à la racine du projet puis faite la commande: "make"

2) Aller dans le répertoire bin qui se trouve à la racine du projet puis faite la commande "./game"

Pour récompiler le projet de manière propre, faite "make clean" puis "make" de nouveau à la racine du projet.


========================
ASSIGNATION DES TOUCHES
========================

Z Déplacement vers le haut
S Déplacement vers le bas
Q Déplacement vers la gauche
D Déplacement vers la droite

C Affiche les statistiques actuelles du joueur (en console):
	
	- Nom
	- Classe
	- Level
	- Expérience actuelle
	- Expérience avant le prochain niveau
	- Vie
	- Force
	- Défense

ESPACE Donne un coup d'épée
